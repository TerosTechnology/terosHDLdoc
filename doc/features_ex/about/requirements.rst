Requirements
==========

All the tools are automatically managed by TerosHDL vscode plugin.
But there are some requisites you have to satisfy to enable the all the functionalities.

.. _Python 3: configured in TerosHDL :doc:`settings <general>`
.. _pyteroshdl: ``pip3 install pyteroshdl``

Simulators
~~~~~~~~~~~~~

Simulators aren't included in TeroHDL. You need to install your simulator and config it properly.
